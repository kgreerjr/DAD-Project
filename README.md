# DAD-Project

A relational database design and implementation project using MySQL, covering schema creation, table relationships, data imports, and advanced SQL queries to analyze customer, order, and RMA data.

## About

- Designed and implemented a normalized relational database schema in MySQL, defining tables with primary and foreign keys to enforce referential integrity across customer, order, and RMA datasets.
- Wrote SQL queries for data imports, multi-table joins, conditional filtering, aggregate functions, and bulk updates to support evolving business reporting requirements.
- Documented the database design and query development process in a structured project report, demonstrating end-to-end database administration from schema design to data analysis.

## Course

DAD 220 - Introduction to Structured Database Environments | Southern New Hampshire University
